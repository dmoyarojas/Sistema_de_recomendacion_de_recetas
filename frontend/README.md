
  # Recipe Finder Platform

  This is a code bundle for Recipe Finder Platform. The original project is available at https://www.figma.com/design/jcKBGSs9ZftxUywwNnvPvW/Recipe-Finder-Platform.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
  

  ## Desarrollo

  Para ejecutar backend hacer 'cd backend' y escribir `python manage.py runserver` se ejecutara en el puerto http://localhost:8000/

  Para ejecutar frontend 'cd frontend' y poner `npm run dev`, se ejecutara en el http://localhost:3000/
